FIX AVANCE DE COPAS

Corregido:
- QF1 + QF2 pasan a Semifinal 1 cuando ambos tienen ganador.
- QF3 + QF4 pasan a Semifinal 2 cuando ambos tienen ganador.
- SF1 + SF2 pasan a la Final cuando ambas tienen ganador.
- La final guarda campeón cuando tiene resultado.
- Si falta un ganador, la ronda siguiente no se completa incorrectamente.

Importante:
En copas no se permite empate. Debe haber ganador para avanzar.
