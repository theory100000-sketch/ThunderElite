SINCRONIZACIÓN CON LA WEB
- Crear, editar o borrar clubes actualiza la web y sus escudos.
- Cambiar presidente o vicepresidentes actualiza la directiva.
- Añadir, quitar o fichar jugadores actualiza la plantilla y el total registrado.
- Las operaciones económicas actualizan el presupuesto y su historial.
- Las sanciones y avisos quedan guardados en data.json.
- Las ofertas solo modifican plantillas y presupuestos cuando se aceptan.
- Los comandos de consulta leen el mismo data.json usado por la web.

INSTALACIÓN
1. Copia .env.example como .env y completa TOKEN, CLIENT_ID y GUILD_ID.
2. Ejecuta ACTUALIZAR_COMANDOS_DISCORD.bat una vez para registrar los 26 comandos.
3. Ejecuta INICIAR_LOCALHOST.bat para iniciar web y bot juntos.
4. Opcional: ejecuta VERIFICAR_COMANDOS_DISCORD.bat para comprobar la instalación.
