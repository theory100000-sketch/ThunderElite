OPTIMIZACIÓN DE CARGA TEL

- Escudos reducidos y comprimidos sin cambiar sus nombres ni rutas.
- Trofeos y recursos gráficos optimizados.
- Caché del navegador para imágenes, CSS y JavaScript.
- Sincronización en tiempo real mediante SSE sin recargar toda la página.
- Sondeo de respaldo reducido para evitar peticiones continuas.
- Eliminados intervalos antiguos que recorrían el DOM cada pocos segundos.
- Imágenes con carga diferida y decodificación asíncrona.
- Actualizaciones públicas agrupadas para evitar solicitudes duplicadas.

El autoguardado y la sincronización entre navegadores siguen activos.
