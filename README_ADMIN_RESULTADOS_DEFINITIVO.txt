PANEL DEFINITIVO RESULTADOS

Abrir:
http://localhost:3000/admin-resultados.html

Este panel:
- inicia sesión admin ahí mismo;
- lee los partidos reales desde data.json;
- guarda directamente en data.json;
- recalcula clasificación y copas;
- fuerza data.json sin caché.

Si algo no se ve en la web principal, entra a admin-resultados.html, guarda, y luego recarga la web principal con Ctrl+F5.
