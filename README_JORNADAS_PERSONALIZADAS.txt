JORNADAS PERSONALIZADAS PARA LIGAS

- En Panel Admin > Competiciones aparece el campo "Nº de jornadas de la liga".
- El administrador puede elegir de 1 a 100 jornadas.
- El campo solo aparece cuando el tipo de competición es Liga.
- Al guardar participantes, el calendario se genera con exactamente ese número de jornadas.
- Si se cambia el número de jornadas, se regenera el calendario.
- Si ya existen resultados, se pide confirmación antes de reiniciarlos.
- La sincronización y el autoguardado siguen activos.
