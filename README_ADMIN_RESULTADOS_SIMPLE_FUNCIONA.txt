ADMIN RESULTADOS SIMPLE QUE FUNCIONA

Se añadió una sección independiente:
#admin-resultados

Uso:
1) Entra como admin.
2) Abre /#admin-resultados o pulsa el enlace "Admin resultados".
3) Elige competición.
4) Elige partido.
5) Mete goles.
6) Guardar resultado.

Este panel no depende de detectar botones visuales ni del bracket, por eso guarda siempre el partido correcto.

También:
- Liga recalcula PJ, PG, PE, PP, GF, GC, DG, PTS.
- Copa no permite empate.
- Copa crea semifinal/final si faltan.
- Copa pasa ganadores automáticamente.
